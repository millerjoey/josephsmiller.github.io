Sum-Product Network Demo

Uncertain Data Explorer

Explore tractable probabilistic queries for mixed data. Select exact, missing, interval/censored, or set-valued evidence, then inspect conditional marginals, relationships, and plausible samples entirely in the browser.

Most dashboards filter observed rows. This demo queries a Sum-Product Network trained offline on finite mixed data. The same training path supports missing and censored observations, so uncertainty can enter the model before you ever start filtering or sampling.

What This Is

Most dashboards filter observed rows. That works when the slice is large and clean, but breaks down when the question gets specific: small subgroups, missing fields, censored values, interval observations, or several partial conditions at once.

This demo is a dashboard over a learned joint distribution. A Sum-Product Network is trained once offline, optionally refined with autodiff, then exported as a static browser model. Because SPNs are tractable, the page can answer many conditional probability and sampling questions quickly without a backend. The same representation can also be used during training, so incomplete records, missing fields, ranges, and censored values do not have to be discarded or forced into single imputed values.
